What is HTML?

HTML is a markup language used to structure web content. It defines elements like headings, paragraphs, links, and media.

Structural Elements

  • header – top section
  • nav – navigation links
  • section – grouped content
  • footer – bottom section

Text Elements

  • h1-h6 – headings
  • p – paragraphs
  • strong – emphasis

Other Elements

  • a – links
  • img – images
  • table – structured data
  • form – user input
